# 30-Day Web3 + AI Bot Project
**Day 7 Milestone** – Live Telegram Crypto Price Bot

A beginner-built Telegram bot that delivers real-time crypto prices, 24h changes, and automatic 5-minute pushes. Built in 7 days from zero.

## Features
- `/price` or `/price ethereum` → instant CoinGecko prices  
- `/subscribe` → true push notifications every 5 minutes  
- Secure token handling with `.env`  
- Built with pyTelegramBotAPI + requests + schedule

## How to Run
1. `pip install pyTelegramBotAPI requests schedule python-dotenv`
2. Create `.env` with `TELEGRAM_TOKEN=your_token`
3. `python day6_bot.py`

## Tech Stack
- Python 3  
- Telegram Bot API  
- CoinGecko free API  
- Git + GitHub for version control
